# HttpServer TLS 支持设计

## 一、需求概述

1. 新增 3 个 Tls 链式函数 —— 分别对应 3 种证书来源，调用即启用 TLS
2. 新增链式函数 `WebRoot()` —— 从 `Start()` 中独立出 WebRoot 参数
3. `Start()` 方法精简为只保留 `Port` 和 `IP`

## 二、设计思路

### 核心理念：调用即启用

不再用一个 `Tls(Boolean)` + 一堆 Optional 参数的大函数，而是拆成 3 个语义明确的链式函数：

- `TlsCertFile` → 证书文件模式（最常用）
- `TlsCertSubject` → Windows 证书存储模式
- `TlsCertMemory` → 内存证书模式（高级）

**调用任意一个 = 启用 TLS**，不调用 = 纯 HTTP。无需布尔参数，无需条件判断。

内部用枚举 `m_TlsMode` 记录用户调用了哪个，`Start()` 据此选择协议和传参，`StopMe()` 将枚举重置为 `TlsNone`。

### TLS 模式枚举

```vb
Private Enum EnumTlsMode
    TlsNone          ' 未启用 TLS（默认）
    TlsModeCertFile  ' 证书文件
    TlsModeSubject   ' Windows 证书存储
    TlsModeMemory    ' 内存证书集合
End Enum
```

## 三、三种证书来源详解

| 证书来源 | 链式函数 | 必要参数 | 可选参数 | 典型场景 |
|---------|---------|---------|---------|---------|
| 证书文件 | `TlsCertFile` | `CertFile` | `Password`, `AlpnProtocols` | Let's Encrypt、自签名、商业证书 |
| Windows 证书存储 | `TlsCertSubject` | `CertSubject` | `AlpnProtocols` | 企业环境，证书在系统证书库 |
| 内存证书集合 | `TlsCertMemory` | `Certificates`, `PrivateKey` | `AlpnProtocols` | 程序动态生成/加载证书到内存 |

### CertFile 格式支持

底层 `InitServerTls` 自动检测：
1. 先尝试 **PFX/P12** 格式
2. 失败则按 **PEM** 格式
3. PEM 支持 `|` 分隔多文件：`"fullchain.pem|privkey.pem"`

### AlpnProtocols

ALPN 协议协商字符串，默认 `"http/1.1"`。HTTP/2 可设为 `"h2,http/1.1"`。

## 四、API 设计

### 4.1 新增成员变量

```vb
' TLS 状态
Private m_TlsMode As EnumTlsMode

' TlsCertFile 参数
Private m_TlsCertFile As String
Private m_TlsPassword As String

' TlsCertSubject 参数
Private m_TlsCertSubject As String

' TlsCertMemory 参数
Private m_TlsCertificates As Collection
Private m_TlsPrivateKey As Collection

' 公共参数
Private m_TlsAlpnProtocols As String

' WebRoot
Private m_WebRoot As String
```

### 4.2 三个 Tls 链式函数

```vb
' 证书文件模式（最常用）
Public Function TlsCertFile( _
    ByVal CertFile As String, _
    Optional ByVal Password As String, _
    Optional ByVal AlpnProtocols As String = "http/1.1") As cHttpServer

    m_TlsMode = TlsModeCertFile
    m_TlsCertFile = CertFile
    m_TlsPassword = Password
    m_TlsAlpnProtocols = AlpnProtocols
    Set TlsCertFile = Me
End Function

' Windows 证书存储模式
Public Function TlsCertSubject( _
    ByVal CertSubject As String, _
    Optional ByVal AlpnProtocols As String = "http/1.1") As cHttpServer

    m_TlsMode = TlsModeSubject
    m_TlsCertSubject = CertSubject
    m_TlsAlpnProtocols = AlpnProtocols
    Set TlsCertSubject = Me
End Function

' 内存证书集合模式（高级）
Public Function TlsCertMemory( _
    ByVal Certificates As Collection, _
    ByVal PrivateKey As Collection, _
    Optional ByVal AlpnProtocols As String = "http/1.1") As cHttpServer

    m_TlsMode = TlsModeMemory
    Set m_TlsCertificates = Certificates
    Set m_TlsPrivateKey = PrivateKey
    m_TlsAlpnProtocols = AlpnProtocols
    Set TlsCertMemory = Me
End Function
```

### 4.3 WebRoot 链式函数

```vb
Public Function WebRoot(ByVal Path As String) As cHttpServer
    m_WebRoot = Path
    Set WebRoot = Me
End Function
```

### 4.4 改造后的 Start 方法

```vb
' 改造前
Public Function Start(Optional Port As Long = 80, Optional WebRoot As String = "", Optional IP As String = "0.0.0.0") As Boolean

' 改造后
Public Function Start(Optional Port As Long = 80, Optional IP As String = "0.0.0.0") As Boolean
    If IsRunning = True Then LastError = "服务器正在运行中": Exit Function

    Set m_oServer = New cTlsReMaster

    With m_oServer
        If m_TlsMode = TlsNone Then
            ' --- 纯 HTTP ---
            .Protocol = sckTCPProtocol
            .Bind Port, IP
            .Listen
        Else
            ' --- HTTPS ---
            .Protocol = sckTLSProtocol
            .Bind Port, IP
            Select Case m_TlsMode
                Case TlsModeCertFile
                    .Listen m_TlsCertFile, m_TlsPassword, , , , m_TlsAlpnProtocols
                Case TlsModeSubject
                    .Listen , , m_TlsCertSubject, , , m_TlsAlpnProtocols
                Case TlsModeMemory
                    .Listen , , , m_TlsCertificates, m_TlsPrivateKey, m_TlsAlpnProtocols
            End Select
        End If
    End With

    Set HttpApp = New cHttpServerApp
    Set HttpSvr = New cHttpServerSvr
    With HttpSvr
        .IP = IP
        .Port = Port
        .WebRoot = m_WebRoot
    End With
    IsRunning = True
    Start = True
End Function
```

### 4.5 StopMe 清理

```vb
' 在 StopMe() 的现有清理逻辑后追加：
m_TlsMode = TlsNone
m_TlsCertFile = ""
m_TlsPassword = ""
m_TlsCertSubject = ""
Set m_TlsCertificates = Nothing
Set m_TlsPrivateKey = Nothing
m_TlsAlpnProtocols = ""
m_WebRoot = ""
```

## 五、使用示例

### 5.1 PEM 证书（Let's Encrypt 最常见）

```vb
Server.TlsCertFile("C:\certs\fullchain.pem|C:\certs\privkey.pem").Start 443
```

### 5.2 PFX 带密码

```vb
Server.TlsCertFile("C:\certs\server.pfx", "my-password").WebRoot("C:\www").Start 443
```

### 5.3 Windows 证书存储

```vb
Server.TlsCertSubject("www.example.com").Start 443
```

### 5.4 内存证书（高级）

```vb
Server.TlsCertMemory(certs, privKey).Start 443
```

### 5.5 纯 HTTP（不调用任何 Tls 函数）

```vb
Server.WebRoot("C:\www").Start 80
```

### 5.6 条件启用（根据配置选不同函数）

```vb
If Config.EnableHTTPS Then
    If Config.CertFromStore Then
        Server.TlsCertSubject Config.CertSubject
    Else
        Server.TlsCertFile Config.CertPath, Config.CertPwd
    End If
End If
Server.WebRoot(Config.WebRoot).Start Config.Port, Config.BindIP
```

### 5.7 HTTP + HTTPS 双端口

```vb
' HTTP
Dim httpSvr As New cHttpServer
httpSvr.WebRoot("C:\www").Start 80

' HTTPS
Dim httpsSvr As New cHttpServer
httpsSvr.TlsCertFile("C:\certs\server.pfx", "pwd").WebRoot("C:\www").Start 443
```

## 六、改造清单

| 文件 | 改动 |
|------|------|
| `cHttpServer.cls` | 新增枚举 `EnumTlsMode`；新增 3 个 Tls 链式函数 + `WebRoot()`；新增 8 个 Private 成员变量；改造 `Start()` / `StopMe()` |
| 其他 HttpServer 文件 | **无需改动** — TLS 对 HTTP 协议解析完全透明 |

## 七、验证要点

1. **HTTPS 基础连通**：浏览器访问 `https://localhost:443` 正常
2. **证书链完整性**：PEM 格式需包含 fullchain（服务器证书 + 中间证书）
3. **ALPN 协商**：Wireshark 抓包确认 ClientHello/ServerHello 中 ALPN 扩展正确
4. **混合部署**：HTTP(80) + HTTPS(443) 双实例同时运行互不干扰
5. **错误处理**：证书文件不存在 / 密码错误 / 证书过期时，`Start` 返回 False 并设置 `LastError`
6. **StopMe 重置**：停止后重新 Start 为纯 HTTP，无需手动清除 TLS 配置
