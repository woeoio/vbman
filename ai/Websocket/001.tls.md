# WebSocket TLS (wss://) 支持设计

## 一、需求概述

- cWebSocketServer：支持 TLS 监听（wss:// 服务端）
- cWebSocketClient：支持 wss:// 连接（当前代码显式拒绝 wss://）

## 二、依赖关系

```
cWebSocketServer ──使用──→ cWinsock ──使用──→ cTlsSocket（TLS 能力来源）
cWebSocketClient  ──使用──→ cWinsock ──使用──→ cTlsSocket（TLS 能力来源）
```

**前提条件：** cWinsock 必须先支持 TLS（见 `ai/Winsock/003.tls.md`），本设计才能实施。

## 三、cWebSocketServer TLS 设计

### 3.1 现状

```vb
' cWebSocketServer.cls L130-132
Set m_ListenSocket = New cWinsock
m_ListenSocket.Protocol = wskTCPProtocol
m_ListenSocket.Listen Port
```

使用 `cWinsock` 作为监听 Socket，TLS 完全透明 — 只要 cWinsock 配置了 TLS，WebSocket 帧的传输自动加密。

### 3.2 TLS 模式枚举

```vb
Private Enum EnumWsTlsMode
    WsTlsNone          ' 未启用（默认，ws://）
    WsTlsModeCertFile  ' 证书文件
    WsTlsModeSubject   ' Windows 证书存储
    WsTlsModeMemory    ' 内存证书集合
End Enum
```

不需要 `TlsModeClient`，因为 WebSocket Server 只有服务端场景。

### 3.3 三个链式函数

```vb
' 证书文件模式（最常用）
Public Function TlsCertFile( _
    ByVal CertFile As String, _
    Optional ByVal Password As String, _
    Optional ByVal AlpnProtocols As String = "") As cWebSocketServer

    m_TlsMode = WsTlsModeCertFile
    m_TlsCertFile = CertFile
    m_TlsPassword = Password
    m_TlsAlpnProtocols = AlpnProtocols
    Set TlsCertFile = Me
End Function

' Windows 证书存储模式
Public Function TlsCertSubject( _
    ByVal CertSubject As String, _
    Optional ByVal AlpnProtocols As String = "") As cWebSocketServer

    m_TlsMode = WsTlsModeSubject
    m_TlsCertSubject = CertSubject
    m_TlsAlpnProtocols = AlpnProtocols
    Set TlsCertSubject = Me
End Function

' 内存证书集合模式（高级）
Public Function TlsCertMemory( _
    ByVal Certificates As Collection, _
    ByVal PrivateKey As Collection, _
    Optional ByVal AlpnProtocols As String = "") As cWebSocketServer

    m_TlsMode = WsTlsModeMemory
    Set m_TlsCertificates = Certificates
    Set m_TlsPrivateKey = PrivateKey
    m_TlsAlpnProtocols = AlpnProtocols
    Set TlsCertMemory = Me
End Function
```

**注意：** WebSocket 的 AlpnProtocols 默认留空（而非 "http/1.1"），因为 WebSocket 有自己的升级机制，通常不需要 ALPN 协商。

### 3.4 Listen 方法改造

```vb
' 改造前
Public Sub Listen(Optional ByVal Port As Long = 8080)
    ' ...
    Set m_ListenSocket = New cWinsock
    m_ListenSocket.Protocol = wskTCPProtocol
    m_ListenSocket.Listen Port
    ' ...
End Sub

' 改造后
Public Sub Listen(Optional ByVal Port As Long = 8080)
    ' ...
    Set m_ListenSocket = New cWinsock
    m_ListenSocket.Protocol = wskTCPProtocol
    
    ' 配置 TLS
    Select Case m_TlsMode
        Case WsTlsModeCertFile
            m_ListenSocket.TlsCertFile m_TlsCertFile, m_TlsPassword, m_TlsAlpnProtocols
        Case WsTlsModeSubject
            m_ListenSocket.TlsCertSubject m_TlsCertSubject, m_TlsAlpnProtocols
        Case WsTlsModeMemory
            m_ListenSocket.TlsCertMemory m_TlsCertificates, m_TlsPrivateKey, m_TlsAlpnProtocols
    End Select
    
    m_ListenSocket.Listen Port
    ' ...
End Sub
```

### 3.5 StopServer 清理

```vb
' 在 StopServer() 末尾追加：
m_TlsMode = WsTlsNone
m_TlsCertFile = ""
m_TlsPassword = ""
m_TlsCertSubject = ""
Set m_TlsCertificates = Nothing
Set m_TlsPrivateKey = Nothing
m_TlsAlpnProtocols = ""
```

### 3.6 新增成员变量

```vb
Private m_TlsMode As EnumWsTlsMode
Private m_TlsCertFile As String
Private m_TlsPassword As String
Private m_TlsCertSubject As String
Private m_TlsCertificates As Collection
Private m_TlsPrivateKey As Collection
Private m_TlsAlpnProtocols As String
```

## 四、cWebSocketClient TLS 设计

### 4.1 现状

```vb
' cWebSocketClient.cls L153-157
If m_IsSecure Then
    RaiseError "Secure WebSocket (wss://) is not yet supported"
    Exit Sub
End If
```

URL 解析已经支持 `wss://`，设置了 `m_IsSecure = True` 和 `m_Port = 443`，只是连接时拒绝了。

### 4.2 改造：自动启用 TLS

当 URL 为 `wss://` 时，自动为 cWinsock 启用 TLS：

```vb
' 改造后
If m_IsSecure Then
    ' wss:// 自动启用 TLS
    m_Socket.Tls
End If
m_Socket.Connect m_Host, m_Port
```

无需链式函数 —— 客户端 TLS 不需要证书，由 URL scheme 决定。

### 4.3 可选：客户端证书支持（mTLS）

极少数场景需要客户端证书（双向 TLS），可后续按需添加：

```vb
' 预留设计（当前不实现）
Public Function TlsClientCert( _
    ByVal CertFile As String, _
    Optional ByVal Password As String) As cWebSocketClient
    m_ClientCertFile = CertFile
    m_ClientCertPassword = Password
    Set TlsClientCert = Me
End Function
```

### 4.4 完整 Connect 改造

```vb
Public Sub Connect(ByVal WebSocketURL As String, Optional ByVal SubProtocol As String = "")
    ' ... 现有 URL 解析逻辑不变 ...
    
    ' 删除原来的 wss:// 拒绝代码
    ' If m_IsSecure Then
    '     RaiseError "Secure WebSocket (wss://) is not yet supported"
    '     Exit Sub
    ' End If
    
    m_SubProtocol = SubProtocol
    m_SecretKey = GenerateWebSocketKey()
    
    m_RecvBuffer.Reset
    m_FragmentBuffer.Reset
    m_IsFragmented = False
    
    m_State = WS_STATE_CONNECTING
    
    Set m_Socket = New cWinsock
    m_Socket.Protocol = wskTCPProtocol
    
    ' wss:// 自动启用 TLS
    If m_IsSecure Then
        m_Socket.Tls
    End If
    
    m_Socket.Connect m_Host, m_Port
End Sub
```

## 五、使用示例

### 5.1 WebSocket Server + TLS（wss://）

```vb
Dim wsSvr As New cWebSocketServer
wsSvr.TlsCertFile("C:\certs\fullchain.pem|C:\certs\privkey.pem").Listen 443
```

### 5.2 WebSocket Client 连接 wss://

```vb
Dim wsCli As New cWebSocketClient
wsCli.Connect "wss://example.com/ws"
```

### 5.3 WebSocket Client 连接 ws://（无变化）

```vb
Dim wsCli As New cWebSocketClient
wsCli.Connect "ws://localhost:8080/ws"
```

### 5.4 PFX 证书

```vb
Dim wsSvr As New cWebSocketServer
wsSvr.TlsCertFile("C:\certs\server.pfx", "mypassword").Listen 443
```

### 5.5 Windows 证书存储

```vb
Dim wsSvr As New cWebSocketServer
wsSvr.TlsCertSubject("ws.example.com").Listen 443
```

## 八、验证要点

1. **wss:// 基础连通**：浏览器 `new WebSocket("wss://localhost:443/ws")` 正常
2. **ws:// 不受影响**：非 TLS 连接行为不变
3. **证书错误处理**：无效证书时客户端应收到 OnError 事件
4. **帧传输正确**：TLS 加密下 WebSocket 帧解析无差异
5. **大数据帧**：加密不影响分片和大数据帧传输
