# 阿里云验证码2.0 详细设计文档

## 一、概述

### 1.1 项目背景

阿里云验证码2.0是阿里云提供的人机验证服务，支持多种验证方式（滑块验证、拼图验证、图像复原验证、无痕验证等），可以有效识别自动化脚本和恶意请求。

本项目旨在为VB6环境提供阿里云验证码2.0的完整解决方案，包括：

- 客户端集成（Web/H5纯前端示例）
- 服务端验证（VB6类库）

### 1.2 技术架构

```
┌─────────────────┐
│  Web/H5 客户端  │  (JavaScript集成，弹出/嵌入式验证)
└────────┬────────┘
         │ 1. 用户验证
         │ 2. 获取CaptchaVerifyParam
         ▼
┌─────────────────┐
│  VB6 服务端     │  (cAliyunCaptcha.cls)
└────────┬────────┘
         │ 3. 调用VerifyIntelligentCaptcha

         ▼
┌─────────────────┐
│ 阿里云验证码API │  (HTTPS REST API)
└─────────────────┘
```

### 1.4 优化说明（v1.1）

本版本基于 HttpClient 的最佳实践进行了优化，主要改进包括：

**代码简化**
- 移除了对 cJson 类的依赖，直接使用 HttpClient 内置的 JSON 处理功能
- 移除了 Scripting.Dictionary 的使用，直接使用 RequestDataJson 构建请求体
- 简化了 HTTP 请求流程，代码量减少约 40%

**功能增强**
- 使用 HttpClient 的 ResponseJson() 方法直接获取响应对象，无需手动解析
- 支持 DebugStart 和 DebugInfo 调试功能
- 更完善的错误处理和资源管理

**使用改进**
- 示例代码更加简洁明了，参考 HttpClient 官方示例
- 添加了更多实用示例（带调试信息的验证、完整HttpClient调用示例）
- 代码风格更加一致，便于维护

**兼容性**
- API 接口保持不变，无需修改现有调用代码
- 仅优化内部实现，对使用者完全透明

### 4.1 弹出式验证设计

#### 文件：www/popup.html

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <title>阿里云验证码2.0 - 弹出式示例</title>

    <!-- 1. 配置全局变量 -->
    <script>
      window.AliyunCaptchaConfig = {
        region: "cn", // 地域：cn=中国内地，sgp=新加坡
        prefix: "xxxxxx", // 身份标识（从控制台获取）
      };
    </script>

    <!-- 2. 动态加载验证码JS -->
    <script
      type="text/javascript"
      src="https://o.alicdn.com/captcha-frontend/aliyunCaptcha/AliyunCaptcha.js"
    ></script>
  </head>
  <body>
    <!-- 3. 预留验证码元素 -->
    <div id="captcha-element"></div>

    <!-- 4. 触发按钮 -->
    <button id="captcha-button">点击验证</button>

    <!-- 5. 验证码初始化 -->
    <script type="text/javascript">
      var captcha;

      window.initAliyunCaptcha({
        SceneId: "xxxxxx", // 场景ID
        mode: "popup", // 弹出式
        element: "#captcha-element",
        button: "#captcha-button",

        // 验证成功回调
        success: function (captchaVerifyParam) {
          console.log("验证成功，参数：", captchaVerifyParam);

          // 调用后端API验签
          fetch("/api/verify", {
            method: "POST",
            headers: { "Content-Type": "application/json" },
            body: JSON.stringify({
              CaptchaVerifyParam: captchaVerifyParam,
              SceneId: "xxxxxx",
            }),
          })
            .then((response) => response.json())
            .then((data) => {
              if (data.Result === true) {
                alert("验证通过！");
              } else {
                alert("验证失败：" + data.VerifyCode);
              }
            });
        },

        // 验证失败回调
        fail: function (result) {
          console.error("验证失败：", result);
        },

        // 获取实例
        getInstance: function (instance) {
          captcha = instance;
        },

        // 样式配置
        slideStyle: {
          width: 360,
          height: 40,
        },

        // 语言
        language: "cn",

        // 超时时间（毫秒）
        timeout: 5000,
      });
    </script>
  </body>
</html>
```

##### 5.1.5.2 验证方法

```vb
'=========================================================================
' 公共方法 - 验证
'=========================================================================

' 验证验证码
'@param CaptchaVerifyParam 客户端验证参数
'@param SceneId 场景ID（可选，建议传入）
'@return 验证结果对象
Public Function Verify(ByVal CaptchaVerifyParam As String, _
                        Optional ByVal SceneId As String = "") As AliyunCaptchaVerifyResult
    On Error GoTo ErrorHandler

    Dim Result As AliyunCaptchaVerifyResult

    ' 参数验证
    If LenB(CaptchaVerifyParam) = 0 Then
        Call RaiseError("CaptchaVerifyParam不能为空")
        Exit Function
    End If

    ' 构建请求URL
    Dim RequestUrl As String
    RequestUrl = m_Endpoint & "/?Action=VerifyIntelligentCaptcha"

    ' 使用HttpClient发送请求（参考HttpClient使用示例）
    With New cHttpClient
        ' 构建JSON请求体
        With .RequestDataJson
            .Item("CaptchaVerifyParam") = CaptchaVerifyParam
            If LenB(SceneId) > 0 Then
                .Item("SceneId") = SceneId
            End If
            ' 可选：查看请求数据
            ' Debug.Print "Request Data: ", .Encode(, 2, True)
        End With

        ' 设置超时时间
        .RequestTimeOut = m_Timeout

        ' 可选：启用调试信息
        ' .DebugStart = True

        ' 设置请求头（如果需要）
        ' .RequestHeaders.Item("X-API-KEY") = "YOUR API KEY HERE"

        ' 设置Content-Type
        .SetRequestContentType(EnumRequestContentType.JsonString)

        ' 发送POST请求
        .SendPost(RequestUrl)

        ' 解析响应
        With .ReturnJson()
            Result.RequestId = .Root("RequestId")
            Result.Result = .Root("Result")
            Result.VerifyCode = .Root("VerifyCode")
            Result.CertifyID = .Root("CertifyID")
            Result.Success = .Root("Success")

            ' 可选：获取错误信息
            If .Root.Exists("Message") Then
                Result.Message = .Root("Message")
            End If
        End With

        ' 可选：查看调试信息
        ' Debug.Print "Debug Info: ", .DebugInfo.Encode(, 2, True)
    End With

    ' 触发事件
    If Result.Result = True Then
        RaiseEvent OnVerifySuccess(Result.VerifyCode, Result.CertifyID, Result.Result)
    Else
        RaiseEvent OnVerifyFailure(Result.VerifyCode, GetVerifyMessage(Result.VerifyCode))
    End If

    Verify = Result
    Exit Function

ErrorHandler:
    Call RaiseError("Verify失败: " & ERR.Description)
End Function
```

#### 7.2.1 重试机制

```vb
' 重试验证
Public Function VerifyWithRetry(ByVal CaptchaVerifyParam As String, _
                               Optional ByVal SceneId As String = "", _
                               Optional ByVal MaxRetries As Long = 3) As Boolean
    Dim RetryCount As Long
    Dim Result As Boolean

    For RetryCount = 1 To MaxRetries
        Result = Me.VerifySync(CaptchaVerifyParam, SceneId)

        If Result = True Then
            VerifyWithRetry = True
            Exit Function
        End If

        ' 等待一段时间后重试
        Call Sleep(1000 * RetryCount)
    Next RetryCount

    VerifyWithRetry = False
End Sub
```

#### 7.2.2 超时处理

```vb
' 带超时的验证
Public Function VerifyWithTimeout(ByVal CaptchaVerifyParam As String, _
                                  ByVal TimeoutMs As Long) As AliyunCaptchaVerifyResult
    Dim StartTime As Long
    Dim Result As AliyunCaptchaVerifyResult

    StartTime = GetTickCount()

    ' 启动异步验证
    Call Me.Verify(CaptchaVerifyParam, "")

    ' 等待结果或超时
    Do
        DoEvents
        If GetTickCount() - StartTime > TimeoutMs Then
            ' 超时处理
            Result.Result = False
            Result.Message = "验证超时"
            VerifyWithTimeout = Result
            Exit Function
        End If
    Loop

End Function
```
