# Modbus 演示程序项目依赖说明

> **重要说明**: 本演示程序依赖 VBMAN.dll,这是一个包含所有 Modbus 功能的 COM DLL 库。

## 核心依赖

### 1. VBMAN.dll 动态链接库

**路径**: `../../../dist/DLL/VBMAN.dll`

**说明**: VBMAN.dll 是一个 COM DLL,包含了所有 Modbus 相关的类库以及网络、工具等功能模块。

**包含的 Modbus 相关类**:
- `cModbus` - 基础 Modbus 类
- `cModbusMaster` - Modbus 主站类
- `cModbusSlave` - Modbus 从站类
- `cByteBuffer` - 字节缓冲区类
- `cWinsock` - Winsock 封装类

**主要枚举类型**:
- `ModbusMasterFunctionCode` / `ModbusSlaveFunctionCode` - 功能码枚举
- `ModbusMasterExceptionCode` / `ModbusSlaveExceptionCode` - 异常码枚举
- `ModbusMasterState` / `ModbusSlaveState` - 状态枚举
- `ModbusMasterProtocolType` / `ModbusSlaveProtocolType` - 协议类型枚举
- `ModbusMasterDefaults` / `ModbusSlaveDefaults` - 默认配置结构体

**在项目中的引用**:

#### MasterDemo.vbp
```
Reference=*\G{...}#1.0#0#..\..\..\dist\DLL\VBMAN.dll#VBMANLIB
```

#### SlaveDemo.vbp
```
Reference=*\G{...}#1.0#0#..\..\..\dist\DLL\VBMAN.dll#VBMANLIB
```

**注意**: `{...}` 是 COM 组件的 GUID,在首次引用后会自动生成。

### 2. DLL 编译说明

**源代码项目**: `src/VBMAN.vbp`

**输出路径**: `dist/DLL/VBMAN.dll`

**编译前准备**:
```bash
1. 打开 src/VBMAN.vbp
2. 确保所有源代码文件完整
3. 编译生成 DLL (File -> Make VBMAN.dll)
4. 输出位置: dist/DLL/VBMAN.dll
```

**DLL 注册**:
- VBMAN.dll 是 ActiveX DLL,编译后会自动注册
- 如果需要手动注册,使用:
  ```
  regsvr32 dist/DLL/VBMAN.dll
  ```

## 安装和配置步骤

### 1. 准备工作

```bash
# 1. 确保已编译 VBMAN.dll
- 打开 src/VBMAN.vbp
- 编译生成 DLL 到 dist/DLL/VBMAN.dll
- 确认 DLL 文件存在

# 2. 检查路径关系
MasterDemo.vbp 路径: src/Demos/Modbus/Master/
VBMAN.dll 路径: dist/DLL/VBMAN.dll
相对路径: ../../dist/DLL/VBMAN.dll
```

### 2. 编译 VBMAN.dll

如果 VBMAN.dll 不存在,需要先编译:

```
1. 启动 Visual Basic 6.0
2. File -> Open Project
3. 选择 src/VBMAN.vbp
4. File -> Make VBMAN.dll
5. 确认输出路径为 dist/DLL/VBMAN.dll
6. 点击 OK 开始编译
7. 编译成功后 DLL 会自动注册到系统
```

### 3. 在 VB6 中打开演示项目

```
1. 启动 Visual Basic 6.0
2. File -> Open Project
3. 选择 MasterDemo.vbp 或 SlaveDemo.vbp
4. 确认没有加载错误
```

### 4. 引用 VBMAN.dll

如果提示找不到 VBMAN.dll:

```
1. Project -> References
2. 勾选 "VBMANLIB" 或浏览找到 DLL
3. 浏览路径: ../../dist/DLL/VBMAN.dll
4. 点击 OK 确认引用
```

### 5. 编译运行

```
1. 按 F5 运行项目
2. 如果出现编译错误,检查引用
3. 确认 VBMAN.dll 已正确引用
4. 成功运行后会看到演示界面
```

## 路径问题解决

### DLL 路径配置

当前 DLL 的相对路径:
```
Reference=*\G{...}#1.0#0#..\..\..\dist\DLL\VBMAN.dll#VBMANLIB
```

路径说明:
```
演示项目位置:
  src/Demos/Modbus/Master/ (或 Slave/)

DLL 位置:
  dist/DLL/VBMAN.dll

相对路径计算:
  ../ (返回到 Modbus/)
  ../../ (返回到 Demos/)
  ../../../ (返回到 src/)
  ../../../../dist/DLL/VBMAN.dll (进入 dist/DLL 目录)
```

### 绝对路径配置

如果相对路径有问题,可以使用绝对路径:

```
Reference=*\G{...}#1.0#0#D:\code\vi\vbmanlib\vbman\dist\DLL\VBMAN.dll#VBMANLIB
```

**注意**: 使用绝对路径不利于项目移植,仅用于调试。

### 系统路径配置(推荐)

为了方便使用,可以将 DLL 注册到系统目录:

```
1. 编译生成 VBMAN.dll
2. 手动注册到系统:
   regsvr32 dist/DLL/VBMAN.dll
3. 在项目中直接通过名称引用,无需指定路径
4. Project -> References -> 找到 VBMANLIB 并勾选
```

## 故障排除

### 问题 1: "User-defined type not defined"

**原因**: 未引用 VBMAN.dll 或 DLL 未注册

**解决方案**:
1. 检查是否已编译 VBMAN.dll
2. Project -> References -> 勾选 VBMANLIB
3. 如果找不到,浏览选择 ../../dist/DLL/VBMAN.dll
4. 确认 DLL 已注册

### 问题 2: "Compile error: Type mismatch"

**原因**: DLL 版本不兼容或未正确引用

**解决方案**:
1. 确认使用 VB6 SP6
2. 确认 DLL 版本与演示项目匹配
3. 重新编译 VBMAN.dll
4. 移除引用后重新添加

### 问题 3: "Can't find project or library"

**原因**: DLL 路径错误或未注册

**解决方案**:
1. 检查 DLL 是否存在于 dist/DLL/ 目录
2. 使用记事本打开 .vbp 文件
3. 检查 Reference 行的 DLL 路径
4. 修正路径后重新打开项目

### 问题 4: 运行时错误 "ActiveX component can't create object"

**原因**: DLL 未注册或初始化失败

**解决方案**:
1. 检查 DLL 是否已注册:
   ```
   regsvr32 /u dist/DLL/VBMAN.dll   (卸载)
   regsvr32 dist/DLL/VBMAN.dll       (注册)
   ```
2. 以管理员身份运行注册命令
3. 确认 VBMAN.dll 编译成功
4. 重新编译演示项目

### 问题 5: "Object variable or With block variable not set"

**原因**: 对象未正确初始化

**解决方案**:
1. 检查 Form_Load 中的初始化代码
2. 确认 DLL 中类定义正确
3. 确认使用正确的创建语句
   ```vb
   Set m_Master = New VBMANLIB.cModbusMaster
   Set m_Slave = New VBMANLIB.cModbusSlave
   ```

### 问题 6: 运行时提示找不到 DLL

**原因**: DLL 路径问题或未注册

**解决方案**:
1. 确认 DLL 位于 dist/DLL/ 目录
2. 注册 DLL 到系统目录
3. 在项目中使用绝对路径引用
4. 或将 DLL 复制到 System32 目录

## 部署要求

### 打包时需要的文件

```
1. 编译后的 EXE 文件
   - MasterDemo.exe
   - SlaveDemo.exe

   - VBMAN.dll (Modbus 功能库)

### 部署注意事项

1. **DLL 注册**: VBMAN.dll 需要在目标系统上注册
   ```
   regsvr32 VBMAN.dll

   - Windows XP SP2 或更高版本
