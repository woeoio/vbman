# MSMQ 安装与配置

## Windows 11/10 安装步骤

### 方法一：通过控制面板安装

1. **打开 Windows 功能**
   ```
   控制面板 → 程序 → 程序和功能 → 启用或关闭 Windows 功能
   ```

2. **勾选 MSMQ 组件**
   展开并勾选以下项目：
   ```
   ☑ Microsoft Message Queue (MSMQ) 服务器
      ☑ MSMQ 服务器核心
      ☑ MSMQ Active Directory 域服务集成（可选，域环境需要）
      ☑ MSMQ 触发器和系统服务（可选）
      ☑ MSMQ HTTP 支持（可选）
      ☑ 多播支持（可选）
   ```

3. **点击确定安装**
   - 等待安装完成
   - 可能需要重启计算机

### 方法二：通过 PowerShell 安装（管理员）

```powershell
# 安装 MSMQ 基础组件
Enable-WindowsOptionalFeature -Online -FeatureName MSMQ-Server -All

# 安装完整功能（包括所有子组件）
Enable-WindowsOptionalFeature -Online -FeatureName MSMQ-Server -All
```

### 方法三：通过命令行安装（管理员）

```cmd
# 安装 MSMQ 服务器
dism /online /Enable-Feature /FeatureName:MSMQ-Server

# 安装 MSMQ 完整功能
dism /online /Enable-Feature /FeatureName:MSMQ-Server /All
```

## Windows Server 安装步骤

1. **打开服务器管理器**
2. **管理 → 添加角色和功能**
3. **功能 → 消息队列**
4. **勾选"消息队列"下所有需要的功能**
5. **安装**

## 验证安装

### 方法一：计算机管理
```
1. 运行 compmgmt.msc
2. 展开"服务和应用程序"
3. 应看到"消息队列"节点
```

### 方法二：服务管理
```
1. 运行 services.msc
2. 查找服务名称："Message Queuing"
3. 确认服务正在运行
```

### 方法三：PowerShell 检查
```powershell
# 检查 MSMQ 服务状态
Get-Service MSMQ

# 检查 MSMQ 功能是否启用
Get-WindowsOptionalFeature -Online -FeatureName MSMQ-Server
```

## 常用队列路径格式

### 本地队列
```
FormatName:DIRECT=OS:.\private$\队列名
.\private$\队列名                    ' 简写形式
```

### 远程队列
```
FormatName:DIRECT=OS:计算机名\private$\队列名
计算机名\队列名                       ' 公共队列
```

### 通过 IP 访问
```
FormatName:DIRECT=TCP:192.168.1.1\private$\队列名
```

## 队列权限配置

### 设置队列访问权限（PowerShell）
```powershell
# 获取队列
$queue = Get-MsmqQueue -Name "队列名"

# 设置权限（示例）
# 注意：详细权限设置需通过 COM 对象或 MMC 界面
```

### 通过 MMC 管理
```
1. 计算机管理 → 消息队列
2. 右键队列 → 属性 → 安全
3. 设置用户/组的权限：
   - 发送消息 (Send Message)
   - 接收消息 (Peek Message / Receive Message)
   - 管理队列 (Manage Queue)
```

## 故障排查

### 问题1：安装选项灰色不可选
**原因**：Windows 版本不支持或已安装
**解决**：检查 Windows 版本，或使用 PowerShell 强制安装

### 问题2：服务无法启动
**排查步骤**：
```
1. 检查事件查看器（Event Viewer）中的 MSMQ 日志
2. 确认依赖服务：Message Queuing Triggers、Message Queuing Access Control
3. 检查防火墙设置
4. 验证网络配置（域环境需正确配置 DNS）
```

### 问题3：无法访问远程队列
**检查项**：
- 目标计算机是否启用 MSMQ
- 防火墙是否开放 MSMQ 端口（默认 1801）
- 网络连通性
- 权限配置

## 卸载 MSMQ

```powershell
# PowerShell 卸载
Disable-WindowsOptionalFeature -Online -FeatureName MSMQ-Server

# 命令行卸载
dism /online /Disable-Feature /FeatureName:MSMQ-Server
```
