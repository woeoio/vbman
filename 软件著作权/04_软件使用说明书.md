## 第一章 软件概述

### 1.1 软件简介

VBMAN（Visual Basic Web Management）是一款基于Visual Basic 6.0开发的网络应用开发框架组件库。本软件为VB6开发者提供现代化的Web开发能力，使传统VB6应用程序能够轻松实现网络通信、Web服务、物联网连接等现代功能。

### 1.2 主要功能

VBMAN框架包含以下核心功能模块：

- **网络通信模块**：HTTP服务器/客户端、WebSocket通信、TCP/UDP Socket
- **工业协议模块**：Modbus TCP/RTU、MQTT协议、NTP时间同步
- **Web开发模块**：路由系统、中间件、会话管理、静态文件服务
- **AI集成模块**：OpenAI兼容接口、流式响应支持
- **工具类库**：数据库操作、JSON处理、加密算法、文件操作

### 1.3 技术特点

1. **多协议支持**：支持HTTP/HTTPS、WebSocket、MQTT、Modbus等多种通信协议
2. **工业级稳定性**：针对工业物联网场景优化，支持高并发连接
3. **现代化能力**：为传统VB6注入Web开发、AI集成等现代功能
4. **易于使用**：提供简洁的API接口，降低开发门槛
5. **文档完善**：包含详细的API文档和使用示例

## 第三章 功能模块介绍

### 3.1 网络通信模块

#### 3.1.1 HTTP服务器（cHttpServer）

功能：在VB6应用程序中创建HTTP服务器，提供Web服务能力。

主要功能：
- 监听HTTP请求
- 路由处理
- 静态文件服务
- Cookie管理
- Session会话管理

#### 3.1.2 HTTP客户端（cHttpClient）

功能：发送HTTP请求，与Web服务交互。

主要功能：
- GET/POST/PUT/DELETE请求
- 请求头自定义
- 表单数据提交
- JSON数据处理
- SSE服务器推送接收

#### 3.1.3 WebSocket通信

功能：提供全双工通信能力，支持实时数据传输。

主要功能：
- WebSocket服务器（cWebSocketServer）
- WebSocket客户端（cWebSocketClient）
- 帧协议处理
- 心跳机制

### 3.2 工业协议模块

#### 3.2.1 Modbus通信

功能：支持工业标准Modbus协议，用于PLC通信。

主要功能：
- Modbus TCP主站/从站
- Modbus RTU主站/从站
- 支持功能码：01/02/03/04/05/06/0F/10
- 自动重连机制

#### 3.2.2 MQTT通信

功能：物联网消息传输协议支持。

主要功能：
- MQTT服务器（Broker）
- MQTT客户端
- QoS 0/1/2消息质量
- 遗嘱消息
- 保留消息

### 3.3 数据库模块

#### 3.3.1 数据库操作（cDatabase）

功能：简化数据库访问操作。

主要功能：
- ADO连接管理
- SQL执行
- 事务处理
- 参数化查询

#### 3.3.2 Redis客户端（cRedisClient）

功能：连接Redis缓存服务器。

主要功能：
- 字符串操作
- 哈希操作
- 列表操作
- 发布/订阅

### 3.4 AI集成模块

#### 3.4.1 AI接口（cAI）

功能：集成大语言模型API。

主要功能：
- OpenAI兼容接口
- 流式响应接收
- 多服务商支持
- 对话历史管理

### 3.5 工具类模块

#### 3.5.1 JSON处理（cJson）

功能：JSON数据解析和生成。

主要功能：
- JSON字符串解析
- 对象序列化
- 数组处理
- 嵌套对象支持

#### 3.5.2 加密哈希（cCryptoHash）

功能：数据加密和哈希计算。

主要功能：
- MD5/SHA1/SHA256哈希
- AES加密/解密
- HMAC计算
- Base64编码

## 第四章 使用示例

### 4.1 创建HTTP服务器示例

```vb
' 创建HTTP服务器实例
Dim httpServer As New cHttpServer

' 设置监听端口
httpServer.Listen 8080

' 设置路由处理
httpServer.Get "/", "HomeHandler"
httpServer.Get "/api/data", "DataHandler"

' 启动服务器
httpServer.Start

MsgBox "HTTP服务器已启动，监听端口8080"
```

### 4.2 发送HTTP请求示例

```vb
' 创建HTTP客户端
Dim httpClient As New cHttpClient

' 发送GET请求
Dim response As String
response = httpClient.Get("https://api.example.com/data")

' 发送POST请求（JSON数据）
Dim jsonData As String
jsonData = "{ ""name"": ""test"", ""value"": 123 }"
response = httpClient.Post("https://api.example.com/submit", jsonData, "application/json")

MsgBox "服务器响应：" & response
```

### 4.3 WebSocket通信示例

```vb
' 创建WebSocket服务器
Dim wsServer As New cWebSocketServer

' 设置监听端口
wsServer.Listen 9000

' 处理连接事件
Private Sub wsServer_OnConnected(clientId As String)
    Debug.Print "客户端连接：" & clientId
End Sub

' 处理消息事件
Private Sub wsServer_OnMessage(clientId As String, message As String)
    Debug.Print "收到消息：" & message
    ' 广播给所有客户端
    wsServer.Broadcast "服务器回复：" & message
End Sub

' 启动服务器
wsServer.Start
```

### 4.4 Modbus通信示例

```vb
' 创建Modbus TCP主站
Dim modbus As New cModbusMaster

' 连接到PLC
modbus.ConnectTCP "192.168.1.100", 502

' 读取保持寄存器（功能码03）
Dim values() As Integer
values = modbus.ReadHoldingRegisters(1, 0, 10)

' 写入单个寄存器（功能码06）
modbus.WriteSingleRegister 1, 0, 100

' 断开连接
modbus.Disconnect
```

### 4.5 MQTT通信示例

```vb
' 创建MQTT客户端
Dim mqtt As New cMQTTClient

' 连接到Broker
mqtt.Connect "broker.example.com", 1883, "client001"

' 订阅主题
mqtt.Subscribe "home/sensor/temperature"

' 发布消息
mqtt.Publish "home/sensor/temperature", "25.5", 0

' 处理接收消息
Private Sub mqtt_OnMessage(topic As String, message As String)
    Debug.Print "主题：" & topic & "，消息：" & message
End Sub
```

### 4.6 JSON处理示例

```vb
' 创建JSON对象
Dim json As New cJson

' 解析JSON字符串
Dim jsonText As String
jsonText = "{ ""name"": ""张三"", ""age"": 30, ""city"": ""北京"" }"
json.Parse jsonText

' 获取值
Dim name As String
name = json.GetValue("name")
MsgBox "姓名：" & name

' 构建JSON对象
json.CreateObject
json.AddProperty "name", "李四"
json.AddProperty "age", 25
json.AddProperty "city", "上海"

' 输出JSON字符串
MsgBox json.ToString
```

### 4.7 AI接口调用示例

```vb
' 创建AI实例
Dim ai As New cAI

' 配置API密钥和地址
ai.ApiKey = "your-api-key"
ai.BaseUrl = "https://api.openai.com/v1"

' 发送消息
Dim response As String
response = ai.Chat("你好，请介绍一下自己")

MsgBox response
```

## 第六章 常见问题

### Q1: 如何注册DLL组件？

A: 使用regsvr32命令注册：
```
regsvr32 VBMAN.dll
```

### Q2: 运行时提示缺少依赖？

A: 确保已安装VB6运行时库和所需OCX控件。可从微软官网下载VB6运行时安装包。

### Q3: HTTP服务器无法启动？

A: 检查端口是否被占用，使用命令查看：
```
netstat -ano | findstr :8080
```

### Q4: 如何获取技术支持？

A: 可通过以下方式获取支持：
- 官方文档：https://www.vb6.pro/docs
- 联系邮箱：215879458@qq.com

### Q5: 是否支持64位系统？

A: VBMAN基于VB6开发，在64位Windows上可运行，但仍是32位应用程序。
